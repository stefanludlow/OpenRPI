Obtain the current room description.

Parameters
[in]roomThe room to examine (i.e. "this->get_description").
Returns
The current room description By default, return the room's description. If there are extra descriptions defined, then pick the closest case and return that instead. If you have an extra description for the current weather & time scenario, return that.

Otherwise fall back on the default room description.

15-12-07: Fixed a long standing bug that didn't display night time weather descriptions, but had to make the function somewhat messier to do so.
